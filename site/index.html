Backflow

Backflow is an automation service that runs coding agents in ephemeral containers on behalf of authorized users. Tasks are submitted via API or SMS and produce pull requests, code reviews, or content summaries.

Code

Submit a prompt, get a pull request. Backflow clones the repo, runs a coding agent, commits, and opens a PR.

Review

Automated PR reviews with inline comments from a coding agent.

Read

Summarize any URL. The reader agent fetches content, drafts a TL;DR, and stores it for semantic search.

Multi-channel

Submit tasks via REST API or SMS. Get status notifications via webhooks or SMS.

SMS Communications

If you have provided your phone number and are authorized to use Backflow, we may send you SMS messages relating to tasks you have submitted (for example, status updates when a task starts, completes, fails, or is cancelled).

  • Opt-in: You are added to the service only after the system operator receives your explicit consent and registers your phone number on the internal allow-list. You initiate the messaging relationship by texting in.
  • Opt-out: You may opt out at any time by replying STOP to any message. You will receive a one-time confirmation that you have been unsubscribed from Backflow task notifications and will not be sent further messages.
  • Help: Reply HELP for assistance, or contact privacy@bbell.dev.
  • Message frequency: Varies depending on how often you submit tasks. Typical volume is 1–10 messages per day.

  • Carriers are not liable for delayed or undelivered messages.

We do not share your phone number with third parties for marketing purposes.
